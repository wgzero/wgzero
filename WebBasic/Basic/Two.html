# HTTP和HTTPS的区别

# I.定义

# 1.http:超文本传输协议，是一个客户端和服务器请求的应答标准，用于从www服务器传输超文本到本地浏览器的传输协议

# 2.https:超文本传输安全协议，是http的安全版本，https由http+SSL/TLS来加密数据包。

# II.两者之间的优缺点

1.http缺点

A.缺点

  • 容易被监听
  • 被伪装
  • 被篡改

2.https的

A.缺点

  • 握手阶段费时
  • SSL证书需要花钱
  • HTTPS连接缓存不如HTTP高效
  • SSL证书通常需要绑定IP
  • 有局限性

B.优点

  • 比http更安全
  • https在使用https进行通信时，利用率SSL进行加密传输、身份认证的网络协议(http+ssl)
