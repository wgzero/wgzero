# JS之基础

# 1.JS操作DOM的几种方式

  • getElementById：通过id来获取dom节点
  • getElementByClassName：通过class来获取dom节点
  • H5新增三个操作dom的方法
    • querySelector
    • querySelectorAll
    • getElementsByTagName

# 2.JS基本数据类型

  • 简单类型

    Number、boolean、string、null、undefined、Symbol、bigInt

  • 复杂类型

    Object、Function、Array

# 3.类型判断

  • typeof：判断基本类型
  • instanceof：判断是否在该对象的原型链上
  • Object.prototype.toString.call

# 4.隐式转换和显示转换

  • 隐式转换

    • ==： JS非严格模式下，会进行隐式转换

    • 数字运算符：

      // +具有拼接的含义
let a = 1 + 'a'; // 1a
// -将表达式的值变成Number类型
let b = 1 - 'a'; // NaN

    • .点： 被隐式的转成为对象

    • Number

# 5.==和===之间的区别

  • 双等表示相等
  • 全等表示值和类型都相等
  • Switch里面的判断是全等：值和类型都要相等的

# 6.null和undefined之间的区别

  • null表示还未存在
  • undefined表示已存在还未赋值
